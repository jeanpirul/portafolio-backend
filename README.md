# Restaurant Siglo 21 - Backend

## Programas necesarios
Para poder utilizar el proyecto en localhost en necesario clonarlo y tener algunos programas necesarios:
- NodeJS v16.10.0 o superior
- IDE de desarrollo de tu comodidad Ej. VS Code
- PostMan para puebas de APIS. (Opcional)
- Git para poder gestionar las versiones.

### Como Clonar
Comando para clonar:
```
git clone https://github.com/jeanpirul/portafolio-backend
```

Ya clonado el proyecto es necesario instalar todas las dependencias con el comando:
```

```

```
npm run dev
```
